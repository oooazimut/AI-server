# Закрытие задачи

## Постановщик закрывает (CREATED_BY == текущий пользователь)

Вызвать `bitrix_api` → `tasks.task.approve`. STATUS станет 5 (завершена). Проверка результата не нужна.

## Исполнитель закрывает (RESPONSIBLE_ID == текущий пользователь)

1. Если результата нет — добавить через `bitrix_api` → `tasks.task.result.add`.
2. Завершить через `bitrix_api` → `tasks.task.complete`.
3. Если TASK_CONTROL=Y, задача перейдёт в STATUS=4 (ждёт контроля) — дальнейшая проверка происходит автоматически через вебхук.

`tasks.task.renew` не использовать.
