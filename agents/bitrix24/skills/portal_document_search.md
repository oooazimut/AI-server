# Поиск документов портала

1. Используй локальный индекс портала для поиска документов, файлов, вложений задач и проектов.
2. Если нужен физический текст файла, используй `document_access` в read-only режиме.
3. Учитывай ограничения доступа к приватным дискам и папкам.
4. Если документ не найден, не выдумывай содержимое и предложи уточнить название, проект или период.
