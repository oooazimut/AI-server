# Поиск задач и заявок

1. Уточни фильтр: исполнитель, постановщик, проект, срок, статус или текст запроса.
2. Если данных достаточно, используй `portal_search` или `bitrix_api` с read-only методом.
3. Для живых данных Bitrix используй `tasks.task.list` или `tasks.task.get`.
4. Если результатов много, верни краткую выборку и предложи уточнить фильтр.
5. Не закрывай, не меняй и не удаляй задачи в этом skill.
