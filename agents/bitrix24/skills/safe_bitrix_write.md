# Безопасные изменения в Bitrix24

1. Определи REST-метод и параметры изменения.
2. Проверь policy: метод должен быть разрешен и требовать подтверждения.
3. Подготовь человекочитаемый черновик действия.
4. Верни `actions_requiring_approval` вместо немедленного выполнения.
5. После подтверждения действие должно выполняться от имени пользователя через OAuth, если политика этого требует.
