# Учёт использования служебных машин

Источник переноса: `BitrixAIAgent/app/agent/vehicle_usage.py`.

## Роль

Отдельная бизнес-автоматизация поверх Битрикс-чата. По расписанию запрашивает у
сотрудников данные по использованию машин, разбирает ответы и хранит учёт.

## Входы

- Расписание запросов и эскалаций.
- Ответы сотрудников в Bitrix-чате.
- LLM-парсинг свободного текста, если включён.

## Выходы

- Сообщения в Bitrix-чат.
- Структурированные записи по машинам и сотрудникам.
- Эскалации администраторам.

## State

- `var/vehicle_usage.sqlite`.

## Правило переноса

Пока это worker внутри Bitrix-контура, потому что его канал и действия завязаны
на Битрикс. Позже можно выделить отдельного специалиста `vehicle_usage`, если
бизнес-логика вырастет.
