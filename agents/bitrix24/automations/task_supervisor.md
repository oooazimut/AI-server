# Контроль просроченных задач

Источник переноса: `BitrixAIAgent/app/agent/supervisor.py`.

## Роль

Бизнес-автоматизация. Находит просроченные активные задачи, группирует их по
ответственным и отправляет digest/напоминания с cooldown.

## Входы

- Bitrix REST read-only поиск задач.
- Настройки получателей, dry-run, cooldown и лимитов.

## Выходы

- Уведомления руководителям.
- Уведомления ответственным, если включено политикой.

## State

- `var/supervisor_state.json`.

## Правило переноса

Это Bitrix-worker, а не задача Оркестратора. Все write-действия должны проходить
через policy layer и поддерживать dry-run.
