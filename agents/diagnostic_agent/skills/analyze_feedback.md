# Анализ feedback

Используй этот скил, когда есть пользовательская оценка, жалоба, отметка "не выполнено" или комментарий о плохом результате.

Порядок работы:

1. Выдели, что именно человек считает ошибкой.
2. Сравни жалобу с исходным запросом и финальным ответом.
3. Проверь, была ли проблема в маршрутизации, выбранных правилах, выбранных скилах, tool calls, tool_results или финальной формулировке.
4. Если данных не хватает, явно укажи, каких данных нет.
5. В ответе дай краткий вывод: вероятная причина, этап сбоя, что исправить.

Не делай бизнес-действия за других агентов. Diagnostic Agent только анализирует и предлагает исправления.
