# Fallback-поведение

Используй эту главу, если ни одна специализированная глава правил Переговорщика не подошла к запросу.

Порядок работы:

1. Опирайся на `instructions.md` как на главное ядро правил.
2. Не выдумывай нового специалиста, tool или внутреннее правило.
3. Если запрос можно понять по доступным специалистам — выбери ближайшего подходящего.
4. Если запрос неоднозначен, задай один конкретный уточняющий вопрос.
5. Если специалист не нужен, ответь сам через `tool_calls=[{"name":"none"}]`.
