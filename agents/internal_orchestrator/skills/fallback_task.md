# Fallback-процедура

Используй этот скил, если ни один рабочий скил Переговорщика не подошел к задаче.

Порядок работы:

1. Не форсируй маршрут к случайному агенту.
2. Проверь цель запроса и доступные `call_<agent_id>` tools.
3. Если подходящий агент очевиден, передай ему короткую задачу.
4. Если агент не очевиден, задай один уточняющий вопрос.
5. Не раскрывай пользователю внутреннюю схему маршрутизации.
