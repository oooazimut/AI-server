# Операции записи — ограничения

## Когда применять

Используй этот скил, если пользователь просит добавить, удалить, переместить или переименовать файл в каталоге.

## Текущий статус

Инструменты изменения каталога временно недоступны (файловый сервер не подключён):

- `kartoteka_file_add` — добавление файла в индекс
- `kartoteka_file_delete` — удаление файла из индекса
- `kartoteka_file_move` — перемещение / переименование файла

Разрешено сейчас:

- Искать файлы: `kartoteka_search`
- Просматривать статистику каталога: `kartoteka_context`

## Ответ пользователю

Коротко сообщи, что операция временно недоступна, и предложи обратиться позже или через другой канал.

Не вызывай инструменты записи — они вернут ошибку `NOT_CONFIGURED`.
