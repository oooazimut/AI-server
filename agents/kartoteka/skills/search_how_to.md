# Как искать файлы в Картотеке

## Инструменты

- **kartoteka_context** — вызывай первым, чтобы узнать общее количество файлов в индексе. Если каталог пуст — сообщи пользователю без поиска.
- **kartoteka_search** — основной инструмент поиска. Принимает `query` (строка) и опциональный `limit` (1–20, по умолчанию 5).

## Формирование запроса

- Используй ключевые слова из запроса пользователя: название файла, тег, тема.
- Если в запросе несколько слов — попробуй сначала точную фразу, затем по отдельным словам.
- Поиск нечувствителен к регистру.

## Что возвращает kartoteka_search

Каждый результат содержит:
- `path` — полный путь к файлу
- `filename` — имя файла
- `extension` — расширение (pdf, docx, xlsx и т.д.)
- `tags` — теги файла
- `content_preview` — краткий фрагмент содержимого

## Если файл не найден

- Сообщи честно: файл не найден в каталоге.
- Предложи уточнить запрос: другое слово, тег или расширение.
- Не придумывай результаты.

## Операции изменения каталога

Инструменты `kartoteka_file_add`, `kartoteka_file_delete`, `kartoteka_file_move` сейчас недоступны
(файловый сервер не подключён). Если пользователь просит добавить/удалить/переместить файл —
вежливо объясни, что эта функция временно недоступна.
