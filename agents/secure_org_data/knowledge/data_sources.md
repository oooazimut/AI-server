# Источники данных

## Когда применять

Применяй эту главу, когда нужно понять, где искать данные.

## Основной источник

Путь к базе задается настройкой:

```text
SECURE_ORG_DATA_ROOT
```

На текущем ПК ожидаемый источник:

```text
C:\Настройки, Инструкции-
```

Нельзя жестко зашивать этот Windows path в код: позже данные будут перенесены на Linux-сервер.

## Метаданные механизма доступа

Метаданные могут лежать отдельно от корня файловой базы и задаются настройкой:

```text
SECURE_ORG_DATA_METADATA_DIR
```

Если путь задан, агент использует оттуда:

- `groups.json`;
- `folder_roles.json`;
- `file_access_overrides.json`;
- `content_index/`.

## Индекс содержимого

Индекс содержимого может задаваться отдельно:

```text
SECURE_ORG_DATA_INDEX_DIR
```

Если отдельный путь не задан, агент ищет индекс в:

```text
SECURE_ORG_DATA_METADATA_DIR/content_index
```

## Что делать, если источник не настроен

Не выдумывай данные. Сообщи, что источник или индекс не настроен.
