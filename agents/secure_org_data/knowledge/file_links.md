# Пути и ссылки на файлы

## Когда применять

Применяй эту главу, когда нужно вернуть человеку источник найденных данных.

## Правило выдачи пути

Для `open` данных путь к файлу можно показать.

Для `protected` данных путь можно показать только если пользователь имеет доступ к этому уровню.

Для `secret` данных на первом этапе путь и содержимое не раскрываются через обычный чат, если механизм доступа не разрешил это явно.

## Формат

Показывай путь коротко и без технического JSON.

Пример:

```text
Источник: C:\Настройки, Инструкции-\Объект\Инструкция.docx
```

## Ограничение

На первом этапе агент не отправляет сам файл и не меняет файл. Он только дает текстовый ответ и разрешенный путь.
