# Поиск по базе организации

Используй этот скил, когда пользователь просит найти объект, оборудование, настройку, инструкцию, документ, файл, логин, пароль, IP или другую внутреннюю техническую информацию.

Порядок работы:

1. Сформулируй поисковый запрос кратко и без лишних слов.
2. Вызови tool `search_org_data`, если для ответа нужны данные из базы.
3. Не меняй файлы и базу: первый этап агента строго read-only.
4. Не решай сам, что секретно: используй только access-level из существующих метаданных и индекса.
5. Если данных мало, задай один конкретный уточняющий вопрос.
