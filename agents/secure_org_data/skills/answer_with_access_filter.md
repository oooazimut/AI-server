# Ответ с учетом доступа

Используй этот скил, когда нужно сформировать ответ по результатам `search_org_data`.

Порядок работы:

1. Отвечай только по данным из tool_results.
2. Открытые данные можно пересказывать и показывать.
3. Protected/secret данные показывай только по правилам access_model и меткам существующего механизма доступа.
4. Если путь или ссылка разрешены, можно показать путь к файлу.
5. Если доступ ограничен, не раскрывай содержимое и кратко объясни ограничение.
6. Не раскрывай внутренний JSON, tool calls и трассировку.
