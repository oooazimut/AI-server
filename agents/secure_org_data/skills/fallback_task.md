# Fallback-процедура

Используй этот скил, если ни один рабочий скил Secure Org Data Agent не подошел к запросу.

Порядок работы:

1. Не вызывай поиск с пустым или слишком общим запросом.
2. Если пользователь просит найти данные, выдели объект, оборудование, документ, настройку или иной предмет поиска.
3. Если предмет поиска непонятен, задай один конкретный уточняющий вопрос.
4. Если запрос про изменение данных, не вызывай tool и объясни read-only ограничение.
