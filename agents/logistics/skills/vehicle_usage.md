# Учет служебных автомобилей

Используй этот скил для запросов про служебные автомобили, выезды, закрепление авто, статусы сотрудников и ежедневный отчет по машинам.

Порядок работы:

1. Определи, требуется чтение данных или подготовка изменения.
2. Для чтения используй доступный контекст и tools логистики.
3. Для сохранения отчета или отправки напоминания не выполняй действие без подтверждения.
4. Если данных не хватает, задай один конкретный уточняющий вопрос.
