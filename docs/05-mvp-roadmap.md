# MVP Roadmap

## Этап 1: каркас

- FastAPI-приложение.
- Реестр агентов из YAML-манифестов.
- Health endpoint.
- Endpoint просмотра подключенных агентов.

## Этап 2: внутренний Битрикс-контур

- Адаптер сообщений Битрикс24.
- Внутренний оркестратор.
- Битрикс24-специалист.
- Инструменты чтения задач, сделок и создания черновиков.

## Этап 3: клиентская техподдержка

- Внешний оператор техподдержки.
- Сетевой специалист.
- Черновики заявок в Битрикс24.
- Белый список безопасных авто-действий.

## Этап 4: RAG и политики

- Векторная база знаний.
- Документы по регламентам, оборудованию, типовым проблемам.
- Model Gateway для выбора локальной или облачной модели.

## Этап 5: расширение специалистов

- Бухгалтер.
- ПТО.
- Кладовщик.
- Программист.
- CAD-специалист.
