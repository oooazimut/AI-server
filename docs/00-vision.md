# Видение проекта

Нужно создать корпоративный сервер ИИ-агентов, который помогает сотрудникам и клиентам через привычные каналы общения.

## Цели

- Сотрудники общаются с внутренним оркестратором через Битрикс24-чат или другой рабочий канал.
- Клиенты общаются с оператором техподдержки через внешний чат, например MAX.
- Узкие специалисты подключаются модульно и отвечают только за свою область.
- Реальные действия выполняются через ограниченные инструменты, а не напрямую моделью.
- Опасные операции требуют подтверждения человека.

## Не цели первого этапа

- Не обучать отдельную модель под каждого специалиста.
- Не строить сложную распределенную систему до появления рабочих сценариев.
- Не давать агентам полный доступ к 1С, Битрикс24, серверам или сетевому оборудованию.
